Node.fwends

A simple Node.js with a few extra features to emphasize knowledge of WebScoket technologies. Leverages the Express framework to provide a fully expressed chat-service including

  • Ability to chat real-time with an unlimited number of users.
  • Aliasing to allow for users to go by a specific name.
  • A feature that notifies all chat participants if another user is typing.
